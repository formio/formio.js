Form

Creates an easy to use interface for embedding webforms, pdfs, and wizards into your application.

new Form( element, form, options )

Creates an easy to use interface for embedding webforms, pdfs, and wizards into your application.

Parameters
Name Type Description
element Object

The DOM element you wish to render this form within.
form Object | string

Either a Form JSON schema or the URL of a hosted form via. form.io.
options Object

The options to create a new form instance.

Name Type Description
readOnly boolean

Set this form to readOnly
noAlerts boolean

Set to true to disable the alerts dialog.
i18n boolean

The translation file for this rendering. @see https://github.com/formio/formio.js/blob/master/i18n.js
template boolean

Provides a way to inject custom logic into the creation of every element rendered within the form.
Examples
import Form from 'formiojs/Form';
			const form = new Form(document.getElementById('formio'), 'https://examples.form.io/example');
			form.ready.then(() => form.build());

<static> embed( embed, options )

Description

Embeds a form into a website. This method is used to automatically include the Form.io styles into the header as well as inject the DOM element used to contain the form embedded into a page.

Form.embed({
			  src: 'https://examples.form.io/example',
			  class: 'formio-form-wrapper'
			}).then((instance) => {
			  instance.on('change', (change) => {
			     console.log('The form has changed!', change);
			  });
			});
Parameters

Embed options to use when embedding.

build() → {Promise.<(Webform|PDF|Wizard)>}

Description

Build a new form. This is a method that performs all three phases of form creation.

  • Render - Which takes the template of the form to create an HTML string of the form.
  • Inject - Which takes the HTML string and then sets the contents of the DOM element with that string.
  • Attach - Which will attach all event listeners, load references, and perform logic on the rendered DOM elements.
const wizard = new Form(document.getElementById('formio'), 'https://examples.form.io/wizard');
			wizard.ready.then(() => {
			  wizard.build().then((instance) => {
			    // Will return a Wizard instance since the form.display is set to "wizard"
			    console.log(instance);
			  });
			});
Returns
  • Returns a promise to either a Webform, PDF, or Wizard display depending on which display property is set on the form JSON.
