title: Multi-Form Workflows layout: vtabs section: examples weight: 21

This demonstration illustrates how the Form Component can be used to create multi-form wizard workflows. Each page within the workflow can decide how to navigate to the next form based upon the user input within the workflow.

<link rel="stylesheet" href="https://cdn.form.io/formiojs/formio.full.min.css">
<script src="https://cdn.form.io/formiojs/formio.full.min.js"></script>
<div id="workflow"></div>

Formio.createForm(document.getElementById('workflow'), 'https://examples.form.io/multiform');

This workflow is a form that uses Form components to bring in multiple forms into a single dynamic workflow. This form may look something like the following.
